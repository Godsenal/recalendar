# RcRangePicker

Date range picker.

## Properties

| Property      | Type         | required | default |
| ------------- | ------------ | -------- | ------- |
| startDate     | Date, string | true     | ''      |
| endDate       | Date, string | true     | ''      |
| onSelectRange | function     | true     |         |

## Basic usage

<Playground>
  <Wrapper>
    {({ startDate, endDate, onSelectRange }) => (
      <RcRangePicker
        startDate={startDate}
        endDate={endDate}
        onSelectRange={onSelectRange}
      />
    )}
  </Wrapper>
</Playground>
